Modern \glspl{IDE} are designed for maximum functionality, yet their high feature density creates significant barriers for \gls{neurodivergent} software developers. Individuals with \gls{ADHD}, \gls{ASD}, or dyslexia often face challenges related to \gls{executive_dysfunction}, including cognitive overload, difficulty initiating tasks, and sensitivity to interruptions in workflow. This research explores the extent to which an \gls{AI} integrated \gls{IDE} plugin can mitigate these challenges.

The project involves the development of a custom VS Code extension featuring three core modules: a Minimalist Toggle to reduce interface-driven distractions, a Task Architect that uses \glspl{LLM} to decompose complex prompts into manageable checklists, and a Visual Styler providing high-contrast, low-strain presets. The effectiveness of the plugin is evaluated through a comparative study using a baseline VS Code "Vanilla" versus the enhanced environment. Success is measured via the \gls{NASATLX} \cite{NASATLX} to quantify mental demand and the \gls{SUS} \cite{Hyzy2022} to assess overall tool effectiveness.

By streamlining the development environment and providing \gls{AI}-assisted task structuring, this project aims to lower the "barrier to entry" for \gls{neurodivergent} developers, allowing them to reallocate cognitive resources from navigating complex tools to solving core programming problems.
